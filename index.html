Projeto Pessoal

Gestor Financeiro Em Desenvolvimento

Meu primeiro projeto pessoal full-stack, marcando uma transicao para novas linguagens e conceitos modernos. Sistema completo de gestao financeira pessoal com controle de despesas, orcamentos, transacoes recorrentes, investimentos e analises estatisticas. Desenvolvido com auxilio de IA (Claude e Codex).

15+ Models
10+ Controllers
8 Stores Pinia
100% TypeScript

Stack Tecnologico

Frontend: Vue 3, Vite, Pinia, Tailwind CSS, Chart.js, Zod
Backend: Node.js, Express, Prisma ORM, JWT, Passport.js
Banco de Dados: MySQL 8.0 com Prisma Client
DevOps: Docker, Docker Compose, Winston Logging

Principais Funcionalidades

Gestao de Transacoes

CRUD completo com filtros, paginacao e organizacao mensal automatica

Transacoes Recorrentes

Agendamento automatico (diario, semanal, mensal, anual) com background jobs

Orcamentos

Limites por categoria com alertas configuraveis (ex: 80% do limite)

Investimentos

Calculo de juros simples e compostos com projecoes de rendimento

Analises e Insights

Deteccao de anomalias via Z-score, tendencias e oportunidades de economia

Autenticacao Completa

JWT, Google OAuth, verificacao de email, reset de senha e multi-sessao

Arquitetura e Aprendizados

  • Full-Stack TypeScript com tipagem estrita em todo o projeto
  • Prisma ORM com 15+ models relacionais e migrations automaticas
  • Vue 3 Composition API com Pinia para state management reativo
  • Seguranca com Helmet, rate limiting, bcrypt e JWT refresh tokens
  • Background Jobs para processamento de transacoes recorrentes
  • Containerizado com Docker Compose (MySQL + Backend + Frontend)
